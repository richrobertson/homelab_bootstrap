# Support

This repository is an opinionated homelab infrastructure template. Support is best-effort and focused on reproducible issues in the repository itself.

## Before Opening an Issue

- Read [readme.md](readme.md)
- Read [PUBLIC_RELEASE_GUIDE.md](PUBLIC_RELEASE_GUIDE.md)
- Read [SECURITY.md](SECURITY.md) for sensitive issues
- Confirm the problem is reproducible from the repository contents rather than a private environment detail

## Use the Right Channel

- Bug reports: open a GitHub issue with the bug report template
- Feature requests: open a GitHub issue with the feature request template
- Security issues: follow [SECURITY.md](SECURITY.md) and do not open a public issue

## Include This Information

- What you expected to happen
- What happened instead
- The Terraform module or path involved
- Relevant command output with secrets removed
- Steps to reproduce the problem
- Versions of Terraform and any affected providers or tools

## Support Boundaries

- Environment-specific consulting is out of scope
- Requests without reproduction details may be closed
- Inactive issues may be closed when no additional information is provided
