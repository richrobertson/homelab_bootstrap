# Networking Stack

Manages homelab network primitives and DNS/network wiring used by the cluster.

## What This Stack Manages

- Network-level primitives and wiring that connect infrastructure and services.
- DNS/network dependencies needed by node provisioning and cluster bootstrap.

## When To Edit

- Update addressing, segmentation, routing assumptions, or DNS integration.
- Add downstream network dependencies required by other stacks.
